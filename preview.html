Proyecto universitario de Inteligencia Artificial

Sistema inteligente de recomendaciones musicales basado en preferencias del usuario

Este sitio presenta un prototipo web que recomienda canciones, artistas o playlists a partir del género, idioma, estado de ánimo y contexto de uso elegido por el usuario. El objetivo es demostrar cómo la inteligencia artificial puede aplicarse a una experiencia digital cotidiana de forma simple, útil y personalizada.

¿Qué hace el sistema?

Analiza preferencias musicales ingresadas por el usuario.
Relaciona esas preferencias con una base de datos musical de prueba.
Genera recomendaciones simuladas con una explicación breve.
Permite visualizar una posible evolución hacia un sistema más inteligente.

Acerca del proyecto

El problema que busca resolver este proyecto es la dificultad que tienen muchos usuarios para encontrar música adecuada dentro de catálogos muy amplios. En plataformas digitales existen miles de canciones, artistas y listas de reproducción, por lo que elegir contenido acorde con los gustos personales puede resultar tardado o confuso. Ante esa situación, el sistema propuesto plantea una solución basada en personalización, donde el usuario ingresa ciertas características y el sistema devuelve sugerencias ajustadas a su perfil.

La inteligencia artificial se utiliza en este proyecto como una lógica de análisis y decisión. Aunque el prototipo mostrado aquí trabaja con reglas y coincidencias entre variables, su enfoque responde al mismo principio de los sistemas inteligentes: interpretar datos del usuario, encontrar patrones de preferencia y generar respuestas de manera automatizada. De esta forma, el proyecto representa una aplicación realista de la IA en un entorno musical y web.

Problema

Sobrecarga de opciones musicales

El usuario puede sentirse perdido entre muchas opciones disponibles y necesita un sistema que filtre el contenido para encontrar música más relacionada con sus gustos y necesidades del momento.

Solución

Recomendación personalizada

El sistema analiza variables como género, idioma, estado de ánimo y contexto de uso para sugerir canciones o playlists coherentes con la información proporcionada.

Funcionamiento

El sistema recibe preferencias del usuario y las compara con una colección de opciones musicales clasificadas. A partir de esa relación, prioriza los resultados que mejor coinciden con el perfil ingresado. En esta versión demo se toman en cuenta cuatro variables principales.

Variable 1

Género

Permite seleccionar el estilo musical preferido, por ejemplo pop, indie, rock, lo-fi o reggaetón.

Variable 2

Idioma

Ayuda a filtrar recomendaciones entre música en español, inglés o mezclas según el gusto del usuario.

Variable 3

Estado de ánimo

El sistema considera si la persona busca algo relajado, alegre, concentrado o con más energía.

Variable 4

Contexto de uso

La recomendación cambia según la actividad actual, como estudiar, entrenar, descansar o conducir.

Demo del sistema

En esta simulación, el usuario elige sus preferencias y el sistema muestra recomendaciones de ejemplo con una breve justificación. La lógica es una representación sencilla de cómo funcionaría el prototipo real.

Parte práctica

La parte práctica del proyecto se enfoca en el desarrollo de un prototipo web. No se trata de un dispositivo físico, sino de una solución digital que usa herramientas de programación, diseño web y almacenamiento de datos para simular el comportamiento de un recomendador musical inteligente.

Herramienta o recurso Uso dentro del proyecto
HTML, CSS y JavaScript Diseño de la interfaz y funcionamiento inicial del prototipo web.
Python y Flask Posible implementación de la lógica del sistema y conexión con la base de datos.
SQLite o MySQL Almacenamiento de canciones, artistas, géneros, idioma y contexto musical.
Visual Studio Code Edición y organización del código del proyecto.
GitHub Pages Publicación gratuita de la versión de demostración del sitio web.

Arquitectura y flujo del sistema

→

El flujo inicia cuando el usuario selecciona sus preferencias. Luego, el motor de recomendación compara esas opciones con la información registrada en la base de datos y devuelve sugerencias musicales. En una versión futura, el sistema podría aprender del historial de uso y mejorar progresivamente la precisión de las recomendaciones.
